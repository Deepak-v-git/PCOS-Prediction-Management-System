# PCOS Prediction and Management System

## 📌 Overview
The **PCOS Prediction and Management System** is an AI-powered healthcare platform that helps in the **early detection and management of Polycystic Ovary Syndrome (PCOS)**.  
It integrates **Wavelet ConvNet (Deep Learning)** for ultrasound scan analysis with a **Flutter mobile app** and **Django backend**, ensuring secure and accessible healthcare support.  

## 🎯 Objectives
- Provide early PCOS detection with high accuracy using Wavelet ConvNet.  
- Offer a centralised platform for patients, doctors, and health experts.  
- Enable lifestyle and treatment management through secure communication and recommendations.  

## 🛠️ Technologies Used
- **Frontend:** Flutter, HTML, CSS, JavaScript  
- **Backend:** Django, Python, Dart  
- **Database:** MySQL  
- **AI/ML:** Wavelet ConvNet, 2D-DWT (Haar), Deep Learning  
- **IDE Tools:** PyCharm, Android Studio  

## 📂 Modules
### 🔹 Admin Module
- Manage doctors, experts, and users.  
- Verify registrations and maintain platform security.  
- Handle complaints and feedback.  

### 🔹 Doctor Module
- Secure login and profile management.  
- Manage availability schedules and appointments.  
- Access patient reports, confirm diagnoses, and chat with users.  

### 🔹 Expert Module
- Health advisors (dietitians, fitness trainers, etc.) provide lifestyle tips.  
- Manage profiles and share customised exercise, diet, and wellness advice.  
- Chat-based guidance with users.  

### 🔹 User Module
- Register and manage personal health profiles.  
- Upload ultrasound images for AI-based PCOS prediction.  
- View prediction results with ~99.7% accuracy.  
- Book doctor appointments and chat with experts.  
- Receive personalised fitness and lifestyle support.  

## ✨ Key Features
- AI-driven PCOS detection (Wavelet ConvNet + 2D-DWT).  
- Role-based secure access (Admin, Doctor, Expert, User).  
- Appointment booking and real-time chat.  
- Lifestyle recommendations from experts.  
- Encrypted data handling and secure login.  

## 📊 Results
- Achieved **99.7% accuracy** in PCOS prediction using deep learning.  
- Improved accessibility of women’s healthcare with a scalable and secure system.
